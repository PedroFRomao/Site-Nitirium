Firmware


Há alguns anos existiam "apenas" microcontroladores e PC´s. Eram mundos semelhantes, entretanto, tratados como se fossem totalmente diferentes, onde raramente se cruzavam, eram complemento um do outro. Hoje a realidade é outra. Existem diversos dispositivos, desde microcontroladores de 8, 16 e 32 bits, DSP´s, à módulos programados em JAVA (EGS5i e G24), FPGA´s, modens com script´s, microcontroladores com velocidades de PC´s e placas com arquitetura de PC´s que realizam trabalhos antes destinados a microcontroladores.


Toda esta diversidade nem sempre significa redução de custos, por isso estamos preparados para dizer qual é a melhor solução para o seu projeto, e nos capacitando sempre para estarmos antecipados ao próximo passo tecnológico e oferecer a melhor solução.


Realizamos projetos em qualquer plataforma e possuímos larga experiência em:


Módulos GMS/3G EGS5/EHS6 (Gemalto)

Módulos GSM LEON (UBLOX)

Placa RASPBERRY PI

PIC ( Microchip )

PSoC ( Cypress )

ARM9 ( ST )

STM32 ( ST )

DSP Black Fin ( Analog Devices )
